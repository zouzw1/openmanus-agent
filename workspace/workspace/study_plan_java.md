# Java（初学者）8 周学习路线图

- 总周期：8 周  
- 每周投入：10 小时  
- 推荐节奏：2 次概念学习 + 1 次编码练习 + 1 次复盘总结  

## 第 1 周：环境搭建与基础语法  
- **学习重点**：  
  - 理解 JDK、JRE、JVM 的关系  
  - 完成开发环境搭建与项目运行  
  - 掌握变量、数据类型、输入输出  
- **本周练习**：  
  1. 编写 HelloWorld 程序并成功运行  
  2. 完成变量声明、类型转换和控制台输入输出程序  
  3. 实现四则运算计算器（命令行交互）  

## 第 2 周：流程控制与数组  
- **学习重点**：  
  - 掌握 if/else、switch 分支结构  
  - 熟练使用 for、while、do-while 循环  
  - 理解一维数组的声明、初始化与遍历  
- **本周练习**：  
  1. 成绩评级系统（A/B/C/D/F）  
  2. 九九乘法表（双层 for 输出）  
  3. 猜数字游戏（随机数 + while 循环）  

## 第 3 周：方法与封装  
- **学习重点**：  
  - 方法定义、调用、参数传递与返回值  
  - 方法重载与基本封装思想  
  - 理解 static 关键字的作用域  
- **本周练习**：  
  1. 编写求最大值、最小值、平均值的工具方法  
  2. 封装一个简易银行账户类（deposit/withdraw/balance）  
  3. 实现字符串反转与字符统计方法  

## 第 4 周：面向对象基础（类与对象）  
- **学习重点**：  
  - 类与对象的概念、属性与行为建模  
  - 构造方法、this 关键字、访问修饰符（public/private）  
  - 对象内存模型（栈与堆初步理解）  
- **本周练习**：  
  1. 创建 Student 类（name/id/gpa）并实例化多个对象  
  2. 实现 Book 类（title/author/price），支持 toString()  
  3. 模拟图书馆借阅系统（Book + Borrower 关联）  

## 第 5 周：面向对象进阶（继承与多态）  
- **学习重点**：  
  - extends 继承机制、方法重写（@Override）  
  - super 关键字、final 修饰符  
  - 多态（父类引用指向子类对象）、向上转型  
- **本周练习**：  
  1. Animal 父类 + Dog/Cat 子类，实现 sound() 多态调用  
  2. 计算器基类 + Add/Sub/Mul/Div 子类（策略模式雏形）  
  3. 打印不同图形面积（Circle/Rectangle/Triangle）  

## 第 6 周：异常处理与集合框架  
- **学习重点**：  
  - try-catch-finally 与 throws 声明  
  - RuntimeException vs Checked Exception  
  - ArrayList、LinkedList、HashMap 基本增删查操作  
- **本周练习**：  
  1. 用户登录验证（捕获空输入、格式错误等异常）  
  2. 学生成绩管理系统（ArrayList 存储 Student 对象）  
  3. 单词频率统计（HashMap<String, Integer>）  

## 第 7 周：IO 流与基础文件操作  
- **学习重点**：  
  - 字节流（InputStream/OutputStream）与字符流（Reader/Writer）  
  - 文件读写（FileInputStream/FileOutputStream, FileReader/FileWriter）  
  - 缓冲流（BufferedReader/BufferedWriter）提升效率  
- **本周练习**：  
  1. 复制文本文件（逐行读取 + 写入）  
  2. 日志记录器（向 log.txt 追加时间戳 + 消息）  
  3. 配置文件解析（读取 key=value 格式并存入 Map）  

## 第 8 周：综合项目实战与复习  
- **学习重点**：  
  - 整合前 7 周知识完成小型 CLI 应用  
  - 代码调试技巧（IDE 断点、变量观察）  
  - Git 基础（init/add/commit/push）与项目结构规范  
- **本周练习**：  
  1. 通讯录管理系统（增删改查 + 文件持久化）  
  2. 复习错题整理（从各周练习中提取 3 个典型 bug 并修复）  
  3. 提交 GitHub 仓库并撰写 README.md

# Java（初学者）8 周练习与验收清单

## 第 1 周验收：  
- 能解释 JDK、JRE、JVM 的区别  
- 能独立编写 HelloWorld、变量演示、四则运算程序  
- 能说清楚基本数据类型及类型转换的使用场景  

## 第 2 周验收：  
- 能使用 if/else、switch、for、while 完成分支与循环练习  
- 能独立完成九九乘法表、成绩评级、猜数字等 3 个练习  
- 能通过调试定位循环边界错误  

## 第 3 周验收：  
- 能封装复用方法并合理设计参数、返回值  
- 能使用 static 修饰工具方法，并区分实例与静态上下文  
- 能基于需求抽象出简单类（如 BankAccount）并测试其行为  

## 第 4 周验收：  
- 能正确创建类、定义属性与方法、使用构造器初始化对象  
- 能使用 this 区分同名参数与成员变量  
- 能通过 new 创建多个对象并验证各自状态独立  

## 第 5 周验收：  
- 能写出继承结构并重写父类方法，体现多态调用  
- 能使用 super 调用父类构造器或方法  
- 能识别编译时绑定（重载）与运行时绑定（重写）  

## 第 6 周验收：  
- 能区分运行时异常与检查型异常，并合理捕获或抛出  
- 能使用 ArrayList 存储对象并完成 CRUD 操作  
- 能使用 HashMap 存储键值对并遍历输出  

## 第 7 周验收：  
- 能用 BufferedReader 读取文件并处理 IOException  
- 能用 BufferedWriter 向文件写入带格式的文本  
- 能结合集合与 IO 实现数据临时持久化（如保存学生列表到文件）  

## 第 8 周验收：  
- 能独立完成通讯录系统（含菜单、功能模块、文件存储）  
- 能使用 Git 初始化本地仓库并提交至少 3 次有意义的 commit  
- 能口头复述 8 周所学核心概念并举例说明其应用场景
