Java（初学者）8 周学习路线图：
- 总周期：8 周
- 每周投入：10 小时
- 推荐节奏：2 次概念学习 + 1 次编码练习 + 1 次复盘总结

第 1 周：环境搭建与基础语法
- 学习重点：
  - 理解 JDK、JRE、JVM 的关系
  - 完成开发环境搭建与项目运行
  - 掌握变量、数据类型、输入输出
- 本周练习：
  1. 编写 HelloWorld 程序并成功运行
  2. 完成变量声明、类型转换和控制台输入输出练习

第 2 周：流程控制与数组
- 学习重点：
  - 掌握 if/else、switch 分支语句
  - 理解 for、while、do-while 循环结构
  - 学习一维数组的声明、初始化和遍历
- 本周练习：
  1. 实现九九乘法表打印
  2. 编写成绩评级程序（A/B/C/D/F）
  3. 开发猜数字游戏

第 3 周：面向对象编程基础
- 学习重点：
  - 理解类与对象的概念
  - 掌握封装、构造方法、this 关键字
  - 学习方法重载与参数传递
- 本周练习：
  1. 创建 Person 类并实现基本属性和方法
  2. 设计 BankAccount 类并实现存取款功能
  3. 实现 Calculator 类支持多种运算重载

第 4 周：面向对象进阶
- 学习重点：
  - 理解继承、方法重写、super 关键字
  - 掌握访问修饰符（public/private/protected）
  - 学习抽象类与接口的基本概念
- 本周练习：
  1. 创建 Animal 类及其子类 Dog 和 Cat
  2. 实现 Shape 抽象类及 Circle、Rectangle 子类
  3. 设计 USB 接口及 Mouse、Keyboard 实现类

第 5 周：异常处理与集合框架
- 学习重点：
  - 掌握 try-catch-finally 异常处理机制
  - 理解 RuntimeException 与 Checked Exception 区别
  - 学习 ArrayList、HashMap 等常用集合类
- 本周练习：
  1. 实现用户注册验证并处理各种异常
  2. 创建学生成绩管理系统使用 ArrayList
  3. 使用 HashMap 实现单词计数器

第 6 周：IO 流与文件操作
- 学习重点：
  - 理解字节流与字符流的区别
  - 掌握 FileInputStream/FileOutputStream、FileReader/FileWriter
  - 学习缓冲流与序列化
- 本周练习：
  1. 实现文件复制工具
  2. 创建日志记录器写入文本文件
  3. 序列化/反序列化学生对象到文件

第 7 周：多线程编程
- 学习重点：
  - 理解进程与线程的概念
  - 掌握 Thread 类与 Runnable 接口创建线程
  - 学习线程同步与死锁避免
- 本周练习：
  1. 实现多线程下载模拟器
  2. 创建银行取款多线程安全程序
  3. 设计生产者-消费者模型

第 8 周：项目实战与复习
- 学习重点：
  - 综合运用所学知识
  - 学习 Maven 项目管理
  - 复习核心概念与常见面试题
- 本周练习：
  1. 开发简易图书管理系统（控制台版）
  2. 实现学生信息管理系统（含文件持久化）
  3. 完成 Java 核心知识点总结文档

Java（初学者）8 周练习与验收清单：

第 1 周验收：
- 能解释 JDK、JRE、JVM 的区别
- 能独立编写 HelloWorld、变量演示、四则运算程序
- 能说清楚基本数据类型及类型转换的使用场景

第 2 周验收：
- 能使用 if/else、switch、for、while 完成分支与循环练习
- 能独立完成九九乘法表、成绩评级、猜数字等 3 个练习
- 能通过调试定位循环边界错误

第 3 周验收：
- 能封装复用方法并合理设计参数、返回值
- 能使用数组存储和处理批量数据
- 能解决数组越界、空指针等常见错误

第 4 周验收：
- 能正确使用继承关系构建类层次结构
- 能区分方法重载与重写的适用场景
- 能合理选择访问修饰符保护类成员

第 5 周验收：
- 能识别代码中潜在的异常并添加适当处理
- 能选择合适的集合类解决实际问题
- 能使用迭代器安全遍历集合

第 6 周验收：
- 能正确选择字节流或字符流进行文件操作
- 能使用缓冲流提高 IO 性能
- 能实现对象的序列化与反序列化

第 7 周验收：
- 能创建并启动多个线程执行并发任务
- 能使用 synchronized 解决线程安全问题
- 能识别并避免简单死锁场景

第 8 周验收：
- 能独立完成一个完整的控制台应用程序
- 能使用 Maven 管理项目依赖
- 能清晰阐述 Java 核心概念及其应用场景
